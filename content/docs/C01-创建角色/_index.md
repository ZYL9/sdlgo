# C01-创建角色

乍一看，角色扮演游戏可能令人生畏。页面后,
页面的规则和信息…你怎么可能全部学会呢?好放松。首先，你不需要学习所有的知识
关于这个游戏。这就是为什么你有这本美丽的书作为你永恒的伴侣和参考。其次，书中很多内容都是关于这个世界、恶棍、英雄和神秘人物的有趣读物，但不是你必须知道的东西。慢慢来，好好享受。

角色的创造相对简单，只需要5个主要步骤:

第一步：八属性与属性加值

第二步：生命值与结构值

第三步：决定灵能

第四步：选择种族和职业

第五步：装备及金钱

第六步：完善角色
